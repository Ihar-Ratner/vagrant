# vagrant
Preparing VM with Vagrant for local work

# Prerequisites
- Install Oracle Virtual Box: https://www.virtualbox.org/wiki/Downloads
- Install Vagrant: https://developer.hashicorp.com/vagrant/install

# Installed Tools
- pipx
- cookiecutter
- azure-cli
- terraform
- go
- docker
- kubectl
- ansible-core
- minikube

# VM Provisioning
`vagrant up --provision`
